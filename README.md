# MineCombat

#### 介绍
We are 宇宙超级无敌大(SuperBig42) team who are taking part in the Netease Leihuo game-making competition. We are making a card combat game inspired by Minecraft.

#### 软件架构
基于Unity


#### 安装教程

1.  克隆此仓库
2.  使用Unity打开本仓库文件夹
3.  等待Unity自动根据package.json信息加载所需库
4.  开始游玩和开发。
